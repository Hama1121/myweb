Command Documentation

Powerful commands to manage security features and protect your server.

!shield help

Displays all available commands and basic usage information.

Usage: !shield help [command]

Example: !shield help antispam

!shield status

Shows the current security status and active protections.

Usage: !shield status

!shield scan

Performs a security scan of the server or specified channel.

Usage: !shield scan [channel]

Example: !shield scan #general

!shield antispam

Manages spam detection and prevention settings.

Usage: !shield antispam [enable/disable] [sensitivity]

Example: !shield antispam enable high

!shield antiraid

Configures anti-raid protection settings and thresholds.

Usage: !shield antiraid [enable/disable] [threshold]

Example: !shield antiraid enable 5

Frequently Asked Questions

Everything you need to know about CyberSecurity Bot.

How do I add CyberSecurity Bot to my server?

Adding CyberSecurity Bot is simple! Click the "Add to Discord" button at the top of this page, authorize the bot with your Discord account, select your server, and confirm the required permissions. The bot will join your server immediately and be ready for configuration.

What permissions does the bot require?

For full functionality, CyberSecurity Bot requires Administrator permissions. However, you can customize permissions for a more limited scope. At minimum, it needs: Manage Server, Manage Roles, Manage Channels, Kick/Ban Members, Manage Messages, and View Audit Log.

Is there a premium version available?

Yes! While the free version offers robust protection, our Premium tier includes advanced features like custom branding, increased rate limits, priority support, and enhanced security options. Visit our Premium page for more details and pricing.

How do I configure the anti-raid system?

Use the !shield antiraid command to configure settings. You can set join rate thresholds, verification requirements, and automated actions. For example, !shield antiraid enable 5 lockdown will trigger a server lockdown if 5+ users join within 10 seconds.

Can I customize the bot's responses?

Yes! With the !shield messages command, you can customize warning messages, notifications, and other bot responses. Premium users get additional customization options including custom embeds and branding.
